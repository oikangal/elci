
AMBA Patch
----------
Bu yama iki dosyayı içerir:
- server/index.js  → /api/admin/unmark endpoint’i eklendi (S/P/Ü kaldırma)
- client/src/pages/App.jsx → Admin kartlarında küçük yuvarlak S/P/Ü + rozetlere tıklayınca UNMARK

Kurulum:
  1) Zip'i proje köküne çıkartın; mevcut dosyaların üstüne yazın.
  2) API tarafını bir kez yeniden başlatın (npm run dev). Frontend otomatik yenilenir.
